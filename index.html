Alaa Al-Hamdan

Jeg heter Alaa Al-Hamdan fra Bergen, Norge. Jeg er erfaren i programmering og systemutvikling, med prosjekter i C++ og C# som inkluderer banksystemer, kryptering, spill og små verktøy.

Min kompetanse omfatter objektorientert programmering, sikkerhet, tilgangskontroll og databasehåndtering med SQL Server. Jeg kombinerer teknisk ferdighet med kreativ problemløsning for brukervennlige løsninger.

Jeg er lidenskapelig opptatt av å utvikle digitale løsninger, lære nye teknologier og skape prosjekter som viser praktisk anvendelse og kvalitet.

Utdanningsbasert banksystem

(C++ konsollapplikasjon)

Utviklet et komplett banksystem i C++ over 7+ måneder med 20 000+ linjer kode og 180 klasser. Systemet har RBAC, sikker innlogging, CRUD-operasjoner, flervalutatransaksjoner, ATM-simulator og full logg/revisjon (Audit Log).

Kompetanse: Avansert C++, objektorientert programmering, sikkerhet, tilgangskontroll og prosjektgjennomføring.

Pizza-bestillingssystem

(C# .NET Framework)

Utviklet kundegrensesnitt i C# med SQL Server for registrering og administrasjon av bestillinger. Inkluderer pålogging, visning og innsending av bestillinger med nødvendig logikk og datavalidering. Fokus på klientsiden.

Spill- og verktøyapplikasjoner

(C# .NET Framework)

Lagde små applikasjoner og spill som kombinerer logikk, grafikk og brukerinteraksjon, inkludert bildebehandling med ColorMatrix, oppgavebehandler, gjettespill, Stein-Saks-Papir, Tic Tac Toe, kalkulator og klokker (digital og analog).

Kryptering og dekryptering av filer

(C# .NET Framework)

Utviklet applikasjon for kryptering og dekryptering av filer (bilder, PDF, tekst, musikk, video) med passord. Bruker kun innebygde krypteringsbibliotek i .NET Framework og et enkelt brukergrensesnitt for sikker og rask behandling.
